## Prerequisites
Make sure you have Node.js (>=18) and npm installed.

## How to run

1. Navigate to the backend folder:
```bash
cd backend
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run start:dev
```

4. The server runs by default at : http://localhost:3000
